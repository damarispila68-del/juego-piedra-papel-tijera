## Descripción general
Este diagrama explica el flujo de un juego de Piedra, Papel o Tijera:
- El jugador elige una opción.
- La computadora selecciona una opción al azar.
- Se compara el resultado.
- Se determina el ganador o empate.
- Se muestra el resultado y se pregunta si se desea jugar de nuevo
## Funcionalidade 
* Interfaz gráfica intuitiva: Presenta botones interactivos que permiten al usuario elegir fácilmente entre piedra, papel o tijera.

* Selección aleatoria de la computadora: El sistema genera automáticamente una elección aleatoria para simular la jugada del oponente.

* Evaluación automática de resultados: Compara las elecciones del jugador y la computadora para determinar si el usuario gana, pierde o empata.

* Marcador en tiempo real: Muestra y actualiza el puntaje acumulado del jugador y de la computadora después de cada ronda.

* Retroalimentación visual: Cambia mensajes, colores o estilos de la interfaz para indicar claramente el resultado de cada partida.

* Control del juego: Permite continuar jugando, reiniciar la partida o comenzar una nueva sesión cuando el usuario lo desee.

* Actualización dinámica de la interfaz: Modifica automáticamente los elementos de la pantalla (textos, puntajes y estilos) sin necesidad de recargar la página.
